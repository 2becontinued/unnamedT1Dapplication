# What is this?

This is a small project Im working on designed to make T1D easier.

# What can this do?

Currently, V1 can:

- Find the carbohydrate ratio for specific foods and calculate the carbs via gram x carb factor (search is buggy) 
- Calculate the insulin needed using a carb to insulin factor ratio.
 
- Offer these functions in a easy-ish to use way.

# Whats included in V2?

Currently V2 is somewhat in the works, Planned features for 0.2.0 are:

- [ ] Pygame integration for sound effects and visuals
- [ ] A simple what-to-do addition that tells you what to do in the event of a low blood sugar
